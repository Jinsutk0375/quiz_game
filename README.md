# quiz_game

A simple and interactive web-based quiz game built using HTML, CSS, and JavaScript.  
The game presents multiple-choice questions, tracks the user's score, and displays results at the end.

---

## Features

- Start screen, quiz screen, and result screen  
- Multiple-choice questions  
- Score tracking  
- Correct/incorrect answer highlighting  
- Progress bar showing quiz completion  
- Final result message based on performance  
- Restart quiz functionality  
- Responsive UI

---

## Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (ES6)** – Logic and interactivity
